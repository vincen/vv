# 网上商城术语说明书

​        本文档定义了网上商城项目的公共术语。在阅读网上商城项目其它文档时，若无特殊定义，应该以此文档中术语为准。

| 术语 | 解释                                                         |
| ---- | ------------------------------------------------------------ |
| 用户 | 泛指公众。                                                   |
| 商户 | 指通过认证的，进驻网上商城的商家。                           |
| 运营 | 维持网上商城正常运行，处理商城运行过程中，因业务产生的事务。 |
| 运维 | 维持网上商城正常运行，处理商城运行过程中，因计算机、网络等产生的事务。 |
